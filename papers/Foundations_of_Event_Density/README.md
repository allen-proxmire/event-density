# Foundations of Event Density

The axiomatic foundational paper of the Event Density framework. Derives the canonical ED PDE from four primitives and seven structural constraints, presents the six original 2D structural analogues (RC/Debye, Barenblatt, Stefan horizon, telegraph horizon, oscillation-modulated PME, temporal tension), extends three of them to 3D in Appendix G, and provides the formal universality, well-posedness, and reproducibility appendices.
