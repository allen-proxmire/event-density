# Event Density: Ontology and Axioms

The philosophical / ontological framing of the Event Density framework. Treats *becoming* — the rate of micro-event production — as the single primitive from which the canonical PDE, its constitutive functions, and the rest of physics emerge.

## Relationship to the Foundational Paper

This paper provides the *interpretive* layer of ED. It argues for the choice of "becoming" as primitive and shows how density, mobility, penalty, and participation are forced by ontological commitments. The companion [Foundational Paper](../Foundations_of_Event_Density/paper.md) provides the *technical* layer: the formal axioms (C1–C7), the canonical PDE, and the derivation of physical correspondences.

Read the Ontology paper first if you want to understand *why* ED makes the choices it makes; read the Foundational Paper if you want to see the formal mathematical consequences.
