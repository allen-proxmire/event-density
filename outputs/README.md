# Outputs

Simulation results generated by the ED-SIM-02 pipeline. These files are not tracked in version control (excluded by `.gitignore`). To regenerate them:

```bash
pip install -e .
python -m edsim certify          # Regenerates all validation outputs
```

Individual analogue outputs can be regenerated via the functions in `edsim/phys/analogues/`.

See [docs/HOW_TO_RUN_ANALOGUES.md](../docs/HOW_TO_RUN_ANALOGUES.md) for detailed instructions.
