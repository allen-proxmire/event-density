# ED-Phys-02: Simulator

## Purpose

Implement the discretized update rule from ED-Phys-01 as runnable simulation code on 1D and 2D lattices.

## Relationship to the ED-Phys Pipeline

- **Depends on:** ED-Phys-01 (Update Rule) for the mathematical update rule, stability constraints, and parameter definitions.
- **Feeds into:** ED-Phys-03 (Cosmological Timeline), ED-Phys-05 (Parameter Sweeps), ED-Phys-06 (Emergent Phenomena).

## Scope

- 1D lattice simulator (radial / homogeneous cosmology)
- 2D lattice simulator (spatial structure formation)
- Boundary condition handling (periodic, absorbing, reflecting)
- Diagnostic and observable extraction
- Visualization utilities
