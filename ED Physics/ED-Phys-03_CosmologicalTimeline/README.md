# ED-Phys-03: Cosmological Timeline

## Purpose

Run the ED simulator forward from initial conditions to produce a complete cosmological history: inflation, structure formation, expansion, and late-time flattening.

## Relationship to the ED-Phys Pipeline

- **Depends on:** ED-Phys-01 (Update Rule) for the evolution equations; ED-Phys-02 (Simulator) for the executable code.
- **Feeds into:** ED-Phys-04 (Physical Analogues) for mapping to observable cosmology; ED-Phys-07 (Analytical Theory) for comparison with closed-form predictions.

## Scope

- Initial condition specification (high-ED, near-uniform states)
- Phase identification (inflation, structure formation, expansion, heat death)
- Observable extraction (scale factor proxy, gradient magnitude, mean density, structure count)
- Epoch boundary detection
