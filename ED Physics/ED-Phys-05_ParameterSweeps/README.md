# ED-Phys-05: Parameter Sweeps

## Purpose

Systematically explore the parameter space of the ED update rule to identify regimes, phase boundaries, and sensitivity.

## Relationship to the ED-Phys Pipeline

- **Depends on:** ED-Phys-01 (Update Rule) for parameter definitions; ED-Phys-02 (Simulator) for execution.
- **Feeds into:** ED-Phys-06 (Emergent Phenomena) for identifying parameter regimes that produce structure; ED-Phys-07 (Analytical Theory) for validating scaling predictions.

## Scope

- Sweeps over alpha, beta, gamma, eta, grid resolution
- Phase diagrams (inflation vs. structure vs. collapse vs. dispersion)
- Stability boundary mapping
- Sensitivity analysis
