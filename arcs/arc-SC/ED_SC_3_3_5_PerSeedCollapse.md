## 1. Purpose

The ED-SC 3.x calibration arc has been consolidated through ED-SC
3.3 rev. 2 + ED-SC 3.0 rev. 3. All four filter-hinge axes have been
mapped: L_ray (scanned at Δ = 0.1 and Δ = 0.02), α_filt (Δ = 0.025),
N_req (canonical quorum levels plus geometric-exclusion limit),
**except ξ itself**, which has been held fixed at `ξ_canonical =
1.7575 lu` in every ED-SC 3.x driver to date.

The cross-scale invariance claim of ED-SC 3.1 rev. 3 asserts that
`f(ρ | ξ, L_ray, α_filt, N_req)` is invariant in the dimensionless
coordinate system `(L_ray/ξ, α_filt, N_req)`. This claim has not
been tested directly: we have mapped the invariant **at one ξ**;
we have not verified it **across ξ**. ξ is the single load-bearing
untested hinge of the entire arc.

**The core question.** If the dimensionless hinge `L_ray/ξ = 1.08`
is held fixed but the absolute ξ is varied, does the motif-
conditioned distribution collapse to a single distribution?
Operationally:

> For each seed i in the canonical 10-seed set, with its own
> 40-snapshot canonical `ξ_i`, compute motifs at
> `L_ray_i = 1.08 · ξ_i` (seed-specific). Do the per-seed
> `(S1_i, S2_i, S3_i)` collapse to the same values within
> bootstrap noise and under the ED-SC 3.0 rev. 3 size-dependent
> flatness threshold?

The existing 10-seed canonical field cache already exhibits a
**35 % natural ξ variation** (ξ_per_seed ∈ [1.62, 2.19] lu from
`xi_canonical.json`). This provides a zero-cost first-pass ξ
variation test: no new field evolution is required, only a
re-execution of motif collection at per-seed `L_ray_i` on the
deterministically-reconstructible cache.

This memo is pre-registration only. No numerics are presented.

## 3. Resonance-window check

Per ED-SC 3.2.6 rev. 2 and ED-SC 3.1 rev. 3, `L_ray` values in the
forbidden windows are structurally disallowed:

- Window A: `L_ray ∈ [2.50, 2.80]`
- Window B: `L_ray ∈ [3.50, 3.90]`

The driver must **hard-fail at launch** if any per-seed L_ray_i
falls in either window. Under the concrete ξ_per_seed table above,
no seed triggers a hard-fail; the guard is defensive in case
future ξ measurements shift.

## 4. Protocol

1. **Cache reuse.** The 10-seed canonical field cache is
   regenerated by re-evolving each seed under
   `r2_grf_falsifier_tests.py` +
   `ED_Update_Rule.ed_step_mobility` with the canonical
   parameters (alpha=0.03, gamma=0.5, noise_sigma=0.0556,
   mobility_exp=2.7, steps=500, size=64, dt=0.05, periodic
   boundary) and the canonical 10-seed set
   `{77, 101, 123, 234, 456, 789, 1011, 1213, 1415, 2021}`.
   Because evolution is deterministic given seeds, this
   re-execution is bit-equivalent to the field cache used in
   ED-SC 3.2.6 v2, ED-SC 3.3, and the ED-SC 3.3 α_filt sub-scan.
2. **Per-seed ξ load.** `ξ_i` is read from
   `outputs/ed_sc_3_1/xi_canonical.json` — **not** recomputed in
   this driver. This ensures the ξ anchor for the per-seed hinge
   is the canonical 40-snapshot value certified in ED-SC 3.1
   rev. 2.
3. **Per-seed motif collection.** For each seed, collect motifs
   under the canonical filter geometry
   `(α_filt = 0.25, N_req = 4)` at the seed's own
   `L_ray_i = 1.08 · ξ_i`. Pool motifs across the seed's own
   fields (single final snapshot of the deterministic evolution,
   as used in the existing drivers' motif-extraction step).
4. **Per-seed statistics.** For each seed compute:
   - `N_i` — motif count
   - `S1_i` — median ρ
   - `S2_i` — IQR of ρ
   - `S3_i` — upper-tail log-slope
   - Bootstrap 16–84 % CIs for S1 and S2 (4000 resamples,
     matching the ED-SC 3.x convention).
5. **Pooled statistics.** Pool motifs across all 10 seeds and
   compute `(S1_pool, S2_pool, S3_pool)` with the same bootstrap
   procedure.
6. **Per-seed deviations.** For each seed:
   - `ΔS1_i = |S1_i − S1_pool| / |S1_pool|`
   - `ΔS2_i = |S2_i − S2_pool| / |S2_pool|`
7. **Cross-seed collapse threshold.** Compute
   `S2_rel_SEM_across_seeds` as the relative standard error of
   `S2_i` across the 10 seeds:
   - `S2_rel_SEM_across_seeds = std(S2_i) / (|S2_pool| · √10)`
   - Size-corrected flatness threshold (ED-SC 3.0 rev. 3 §3):
     `flat_thresh = max(0.20, 2 · S2_rel_SEM_across_seeds)`
8. **ξ guardrail.** Inherit from ED-SC 3.2.6 rev. 2: the per-seed
   ξ guardrail pass-rate is read from `xi_canonical.json` and
   must be ≥ 8/10 within 20 % of `ξ_canonical = 1.7575 lu`. This
   is a transparent inheritance — no re-measurement.

---

## 5. Verdict criteria

A single three-way verdict is emitted:

- **`Collapsed`** iff:
  - `ΔS1_i < flat_thresh` for all 10 seeds, AND
  - `ΔS2_i < flat_thresh` for all 10 seeds, AND
  - ξ guardrail ≥ 8/10 pass, AND
  - no per-seed `L_ray_i` falls in Window A or Window B.
- **`Broken-collapse`** iff:
  - ξ guardrail passes, AND
  - at least one seed has `ΔS1_i ≥ flat_thresh` or
    `ΔS2_i ≥ flat_thresh`.
  The verdict records which seed(s) fail and by how much,
  because the pattern of failure (single-outlier vs. systematic)
  distinguishes "one anomalous seed" from "invariance does not
  hold."
- **`GuardrailFailure`** iff:
  - ξ guardrail < 8/10 pass, OR
  - any per-seed `L_ray_i` intersects a forbidden resonance
    window.
  In this case no distributional verdict is emitted; the guardrail
  failure must be resolved before the test can be re-run.

**Interpretation of `Broken-collapse`.** Per-seed failure is the
strongest possible structural finding this test can produce — it
would indicate that the distributional invariance claim of
ED-SC 3.1 rev. 3 is wrong or incomplete, and that the entire
ED-SC 3.x arc needs re-examination. `Collapsed` is the null
hypothesis of the arc as written.

**Scope note.** `Collapsed` verifies invariance across a 35 %
natural ξ variation (ξ ∈ [1.62, 2.19]). It is **not** a universal
cross-scale claim — a broader ξ range requires ED-SC 3.4 (scope
TBD, not this memo). But 3.3.5 is the first direct test of the
invariance claim on any ξ variation, and is a prerequisite for
any claim beyond one-point certification.

## 7. Consequences

- **If `Collapsed`:** the ED-SC 3.x arc is validated on its
  primary claim at 35 % natural ξ variation. ED-SC 3.4 can be
  narrowed from a 4-axis sweep to a deliberate 1D ξ-scan
  covering an order of magnitude (initial-condition amplitude,
  mobility prefactor, or lattice rescaling as the ξ control).
  GR-SC 1.0+ can reference ED-SC 3.3.5 as substantive evidence
  that the distributional substrate is ξ-invariant, not merely
  certified at one ξ.
- **If `Broken-collapse`:** ED-SC 3.1 rev. 3's cross-scale claim
  is wrong or incomplete. The arc must be re-examined before
  ED-SC 3.4 or GR-SC 1.0 can proceed. Candidate remediations —
  refining the dimensionless coordinate (perhaps `L_ray/ξ` is
  not the correct dimensionless hinge), conditioning on a
  second ξ-scale, or restricting the invariance claim to a
  subset of the seed ensemble — are themselves new memo
  scope, not patches to existing memos.
- **If `GuardrailFailure`:** resolve the guardrail issue before
  re-running. The current `xi_canonical.json` is known to pass
  9/10 and no seed L_ray falls in Window A under the concrete
  table of §2; this outcome is defensive, not expected.
