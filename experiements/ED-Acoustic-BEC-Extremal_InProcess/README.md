# ED-Acoustic-BEC-Extremal — BEC Engineered Extremal Horizon (Stub, In Process)

**Target prediction.** ED acoustic metric + P4 mobility-collapse → engineered interior-extremum BEC horizon is extremal (`κ = 0`, no Hawking-analogue phonon pair emission).

**Origin.** Experiment 3 of the ED acoustic-analogue experimental program. Highest-profile follow-up after EIT + optical-pulse within-apparatus differentials. See [`theory/ED_Acoustic_Analogue_Experimental_Program.md`](../../theory/ED_Acoustic_Analogue_Experimental_Program.md) §5.3, §8.1.

**Observable.** Density-density correlation function `G(x, x')` peak amplitude at the Hawking-pair null-ray locus; phonon momentum-distribution thermal tail.

**Prediction.** Correlation peaks and thermal tail suppressed in extremal configuration relative to monotonic (Steinhauer-standard) configuration.

**Platform.** Elongated 1D Rb-87 BEC with custom trap + flow engineering to produce a smooth second-order zero of `v²(x) − c_s²(x)` at an interior point.

**Status.** **Stub protocol, 2026-04-22.** Full protocol blocked on:

1. Moving-background ED acoustic-metric theory extension (current derivation is `v_0 = 0` only).
2. Trap + flow co-design validated in a partner group's Gross-Pitaevskii numerical model.
3. Differential-configuration protocol (monotonic-vs-extremal within the same apparatus or same run).

**Why a stub now.** The BEC platform is the highest-profile analogue-gravity venue (Steinhauer-group Nature-level publications), but the technical + theoretical overhead is larger than the cheaper EIT / optical-pulse paths. Stub captures the concept without committing execution resources until the within-apparatus differentials at lower cost have narrowed the parameter space.
