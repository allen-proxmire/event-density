# ED-Acoustic-EIT-Extremal — Slow-Light Extremal-Horizon Differential (In Process)

**Target prediction.** ED acoustic metric + Canon P4 mobility-collapse → interior-maximum horizons are extremal (`κ = 0`, `T_H = 0`).

**Origin.** Experiment 1 of the ED acoustic-analogue experimental program — top priority for within-apparatus distinguishing power. See [`theory/ED_Acoustic_Analogue_Experimental_Program.md`](../../theory/ED_Acoustic_Analogue_Experimental_Program.md) §5.1, §8.1.

**Observable.** Thermal-emission-rate ratio `R_E / R_M` between extremal (E) and monotonic (M) control-field configurations in the same EIT atomic cell.

**Prediction.** `R_E / R_M < 0.1` (PASS); `> 0.5` (FAIL).

**Platform.** Spatially shaped EIT control field on a warm Rb vapour or cold Rb ensemble, toggled by SLM between monotonic and extremal `v_g(x)` profiles.

**Methodological strength.** Within-apparatus differential — same cell, same detection chain, same photon budget, toggled configurations. Eliminates most systematic drift between (M) and (E) measurements.

**Honesty statement.** The `κ = 0 at smooth max` result is a kinematic property of any acoustic metric, not ED-specific in isolation. This test discriminates *between configurations within one acoustic-metric framework*, which ED adopts. A PASS supports ED's P4-is-extremal reading consistent with the kinematic prediction; a FAIL would rupture either the acoustic-metric identification or ED's extremal-horizon mapping.

**Cost / timeline.** Moderate. If a collaborating EIT group has an SLM-shaped control-field apparatus, ~3–6 months end-to-end. Outreach + apparatus-modification + data collection + analysis.

**Status.** Protocol drafted 2026-04-22. No collaboration contacted. Feasibility review by a domain expert is the recommended first step before any outreach.
