# Reference Figures

This directory contains reference Atlas figures generated by the full reproducibility pipeline.

After running `python reproducibility/run_all.py`, the invariant figures are saved to `output/figures/invariants/`. Copy representative figures here for version-controlled reference:

```bash
# Example: copy the low-mode collapse figure
cp output/figures/invariants/low_mode_collapse/modal_collapse_IC1.png \
   reproducibility/figures/

# Example: copy the certificate
cp output/atlas/ED_Consistency_Certificate.txt \
   reproducibility/figures/
```

These reference figures serve as visual baselines for cross-environment reproducibility checks.
