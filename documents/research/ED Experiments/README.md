# ED Experiments

The empirical program of the Event Density framework.

This folder contains the falsifiable predictions, experimental designs,
and completed tests that connect the ED architectural ontology to
measurable physical outcomes. Every prediction in this folder traces to
the seven canonical principles (P1–P7) through the mathematical
infrastructure of the Rigour Paper (Appendices C and D). Every prediction
has a stated falsification condition.

---

## Role in the ED Program

The ED Architecture defines the structural principles. The Rigour Paper
proves they are mathematically sound. The ED-SIM pipeline confirms they
are computationally self-consistent. This folder asks the next question:
*what do the principles predict about the physical world, and how do we
test those predictions?*

The experimental program is organized around **ED-complexity** — the
gradient content of the density field — which serves as the bridge between
the abstract PDE architecture and laboratory observables. ED-complexity
orders physical systems not by mass or size but by the spatial
inhomogeneity of their density fields, and the architecture predicts that
this ordering governs decoherence, coherence limits, transport thresholds,
halo formation, and spectral saturation.

## Completed Test

### SPARC-Based Temporal Tension Test
**`04.5_Temporal Tension_SPARC‑Based Test.pdf`** and **`04.5_SPARC_Data and Chart.xlsx`**

The first empirical test of an ED prediction. ED predicts that the
discrepancy between observed rotation curves and baryonic mass predictions
arises from temporal halos generated by baryonic activity — not from dark
matter. This test uses the SPARC dataset (46 dwarf galaxies) to check
whether the rotation-curve residuals separate by dynamical activity, as
ED requires.

**Status: Confirmed.** The activity-dependent separation predicted by ED
is present in the SPARC data. This is the one completed test of the
nineteen predictions.
